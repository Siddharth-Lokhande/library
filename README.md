# SiddharthLokh Library
A custom Java utility library with helpful functions for Strings, Math, and Dates.
## Installation
Add JitPack repository to your `pom.xml`:
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
Add the dependency:

<dependency>
    <groupId>com.github.Siddharth-Lokhande</groupId>
    <artifactId>library</artifactId>
    <version>1.0.0</version>
</dependency>
